🧩 Hexo란?

  • 빠르고, 마크다운을 이용해서 글을 쓸 수 있고, GitHub Pages에 쉽게 배포할 수 있어요.
  • Node.js 기반이며 테마와 플러그인도 다양하게 쓸 수 있습니다.
