✨ 오늘 배운 CSS 정리

웹 디자인의 핵심인 CSS 개념을 예제와 함께 정리합니다.

📦 박스 모델(Box Model)

모든 HTML 요소는 기본적으로 박스 형태로 구성됩니다.

  • margin: 바깥 여백
  • border: 테두리
  • padding: 안쪽 여백
  • content: 내용

🎯 position 속성

값 설명
static 기본 위치
relative 원래 위치 기준으로 이동
absolute 부모 중 position이 지정된 요소 기준
fixed 화면 고정
sticky 스크롤에 따라 고정/해제 전환

🧱 display 속성

  • block: 블록 요소 (div, p, h1 등)
  • inline: 인라인 요소 (span, a)
  • inline-block: 인라인처럼 배치 + 크기 지정 가능
  • flex: 유연한 레이아웃 (부모에 적용)
  • grid: 격자형 배치 (부모에 적용)

🌈 기타 속성들

padding, margin

border

box-shadow

🐱 예제: 고양이 만들기 요약

  • .cat: 고양이 전체 컨테이너
  • .face: 얼굴 (원형)
  • .ear: 귀 (회전된 반원)
  • .eye: 눈 (검정 원)
  • .nose: 코 (중앙 정렬된 원)
  • .whisker: 수염 (가로선)

💡 탭 UI 구현 요약

  1. input[type="radio"]로 탭 상태 감지
  2. label로 탭 버튼 디자인
  3. :checked + label로 선택된 탭 스타일링
  4. ~ 선택자와 display: block으로 내용 표시

📝 마무리

CSS는 레이아웃과 디자인을 위한 핵심 언어입니다.
하나씩 구조를 눈으로 보면서 연습하는 것이 중요합니다!
